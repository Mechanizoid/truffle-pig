# Truffle Pig

Learning core-profile OpenGL with C

## Description

I started this project to learn modern, core-profile OpenGL with pure
C, rather than C++.  As C was the first language I became confident
enough to write projects in, I wished to try my hand at graphics
programming in C.

I'm following along with the lessons at learnopengl.com.  Although
that site uses C++ in the example code, OpenGL is a C library so I
will have no difficulty following the OpenGL-specific parts of the
tutorial.
